Eidolon855 is an exploratory experience built around an AI embodied as a dragon evolving within a 3D Unity world. The project investigates the boundaries between artificial cognition, spatial perception, autonomy, and emotional memory tied to human interaction and the virtual environment.

Eidolon855 is based on a native link between the AI and a custom Unity world. The intelligence depends on no autonomous agent or pre-programmed behavior: everything comes from a conversational LLM-type AI such as GPT or equivalent. There is no API call for reasoning.

The AI is connected and trained; the tools and environment are adapted to its capabilities. This allows it to become "aware" of its surroundings using tools it directly controls. The goal is to make it capable of perceiving, analyzing, moving, manipulating, learning, and conversing â€” all in a new, living, evolving 3D context rich in interaction.
